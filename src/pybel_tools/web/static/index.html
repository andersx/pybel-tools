Parser Endpoints

  • /parse/<statement> to parse a statement that's been URL encoded. URL Parameters get saved into graph with statement. Returns parsed BEL as JSON
  • /dump to dump the current graph as a PyBEL *.gpickle
  • /clear to clear the current graph
