Introduction

This web application provides an environment for exploring knowledge assemblies encoded in Biological Expression Language (BEL) to elucidate disease-specific, mechanistic knowledge. It is built on top of the Python software libraries, PyBEL and PyBEL Tools to take advantage of their parsing, validation, data interchange, and analytical functions. From this page, you can begin to explore the networks stored in the knowledge base stratified by the source BEL script, by nodes and other biological entities of interest, or by certain authors and publications of interest.

Parser

Parse and validate a BEL script with the option to store for later querying and analyses. Automatic network analytics are run and presented after parsing. Additionally, pre-parsed BEL scripts can be uploaded from Pickle files. In red is the currently experimental asynchronous parser.
